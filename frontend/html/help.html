Help & User Guide

This guide will help staff members use the reservation system correctly.

1. Login

  • Open the Login page.
  • Enter your username and password.
  • Click the Login button.
  • If details are correct, you will be redirected.

2. Add New Reservation

  • Go to the Reservation page.
  • Fill in all guest details.
  • Select room type and dates.
  • Click "Add Reservation".

3. View Reservation

  • Enter the reservation number.
  • Click the Search button.
  • Guest details will be displayed.

4. Print Bill

  • Select the reservation.
  • Click "Generate Bill".
  • Total amount will be calculated automatically.

5. Validation Rules

  • All fields must be filled.
  • Dates must be valid.
  • Contact number must contain numbers.

6. Exit System

  • Use the Logout or Exit option.
  • Always close the browser after use.

7. Common Errors

  • Invalid login → Check username/password.
  • Empty fields → Fill all inputs.
  • Wrong date → Select correct dates.

Support

If problems continue, contact the system administrator.
